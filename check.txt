để lấy file trên github về : git clone link_http

git status : để xem trạng thái của git

git add .
git commit -m "viet nhuwng thu mik muoosn de commit"
git push origin main

để đẩy từ pc lên: 
git init
git status
git add .
git commit -m ".."
git remote add origin http// ...... nơi mới tạo ấy
git push origin master  

cách pull về lỗi: git reset --hard

CÁCH TẠO 1 SSH KEY để clone lên github: mở git bash :ssh-keygen
git branch -M 'main': DÙNG ĐỂ CHUYỂN NHANH NẾU ĐÓ LÀ MASTER 
git push -u origin 'main'


NẾU MÀ: git remote -v đã có và muốn xóa đi gõ lệnh, git remote rm origin 
ĐỀ TẠO BRAND MƠI: git checkout -b dev
